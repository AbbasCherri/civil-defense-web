# Civil Defense Emergency Management System – Web Frontend

Web application for dispatchers, administrators, and decision makers to manage incidents, resources, and analytics in real time.

## Overview

- **Dispatchers:** Live incident board, map view, team assignment.
- **Administrators:** User management, resource tracking.
- **Decision Makers:** Dashboard with KPIs, charts, and PDF reports.

Built with **FRONTEND FRAMEWORK** and consumes the backend API.

## Prerequisites

- Node.js 18+
- npm or yarn
- Backend API running (see `civil-defense-backend`)
